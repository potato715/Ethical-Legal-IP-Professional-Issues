# Ethics Impact Assessment

## Stakeholders
People affected by this project:
- Students — the main users of this system
- Teachers — who review and manage the data
- School administration — who oversee the project
- Parents — whose children's data may be involved

## Ethical Risks
| Risk | Description | How We Address It |
|------|-------------|-------------------|
| Privacy | User data could be exposed | Passwords are protected, .env used |
| Unauthorized access | Wrong people could log in | Authentication added |
| Data misuse | Student info could be misused | Only admin can access sensitive data |
| Bias | System could treat users unfairly | Same rules apply to all users |

## Summary
This project was built with care to protect all users,
especially students, and to follow ethical standards
expected in a school environment.
