# Privacy Note

## What data does this project collect?
- Username (used for login only)
- Password (not stored in plain text)

## What data do we NOT collect?
- No personal ID numbers
- No addresses or contact details
- No financial information

## How is data protected?
- Sensitive values are stored in .env files
- Passwords are never shown in the code
- Only authorized users can log in

## User Rights
Users have the right to:
- Know what data is collected
- Request their data be removed
- Use the system safely and securely
